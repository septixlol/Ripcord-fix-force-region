# Ripcord-fix-force-region

Here's a .bat file that will fix the forced region (which restarts voice chat).

## 🇫🇷 Tuto: Désactiver le "Force Region" avec IPv4

1. **Ouvre le script**
   Double-clique sur `Plouf Priority Manager.bat` Si une fenêtre s’ouvre demandant les droits administrateur accepte.

2. **Regarde le menu**
   Tu verras trois choix :

   * `1` : Priorité IPv4 → désactive le "force region"
   * `2` : Priorité IPv6 → remet le système par défaut
   * `3` : Quitter

3. **Active IPv4**
   Tape **1** et appuie sur **Entrée**.

   * Si c’est déjà actif → le script te le dira.
   * Sinon → il change la configuration pour que IPv4 soit prioritaire.

4. **Redémarre le PC**
   Le script te proposera de redémarrer pour appliquer les changements.

   * Tape **Y** pour redémarrer maintenant ou **N** pour plus tard.

5. **Vérifie**
   Après le redémarrage, relance le script : le statut doit indiquer **IPv4 Priority**.

Voilà, le "force region" est désactivé et IPv4 est prioritaire. 

**⚠️: Si tu rencontres des problèmes dus au changement de priorité, remets la priorité sur IPv6 après avoir fini d’utiliser Ripcord.**

---

## 🇬🇧 Tutorial: Disable “Force Region” with IPv4

1. **Open the script**
   Double-click `Plouf Priority Manager.bat`. Accept the admin prompt if it appears.

2. **Check the menu**
   You’ll see three options:

   * `1`: IPv4 Priority → disables “force region”
   * `2`: IPv6 Priority → restores default settings
   * `3`: Exit

3. **Enable IPv4**
   Type **1** and press **Enter**.

   * If it’s already active → the script will tell you.
   * Otherwise → it updates the system to prioritize IPv4.

4. **Restart your PC**
   The script will suggest a restart to apply changes.

   * Press **Y** to restart now or **N** to do it later.

5. **Check**
   After reboot, open the script again. It should show **IPv4 Priority**.

Done, IPv4 is now prioritized and “force region” is disabled. 

**⚠️: If you encounter problems due to the priority change, switch back to IPv6 priority after finishing using Ripcord.**
