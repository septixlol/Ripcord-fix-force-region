# Ripcord-fix-force-region

Salut a tous voici un .bat qui va faire en sorte de fix le force région (qui relance la vocal)

tuto:

- Lance le bat 
- Il va te demander de le mettre en admin met oui
- Sélectionne le premier choix
- Relance ton pc

Maintenant tu n'auras plus de probleme sur le force region.

**Attention le script désactive ipv6 donc peut bloquer certaines fonctionnalité comme des sites ou quoi tu peux le réactiver directement depuis le .bat et relancer ton pc**
